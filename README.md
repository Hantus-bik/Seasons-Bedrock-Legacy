# Minecraft Seasons Bedrock (Legacy)

This repository contains the original Seasons Addon for Minecraft Bedrock Edition.

## Project Status

Archived / Legacy

The project was later redesigned into a different direction due to technical limitations involving:

* Bedrock biome override behavior
* Visual subpack synchronization
* Automatic season cycling
* Runtime weather integration
* Script API limitations

## Features

* 12 seasonal phases
* Biome visual overrides
* Seasonal weather system
* Temperature simulation
* Script API experiments
* Resource Pack seasonal themes
